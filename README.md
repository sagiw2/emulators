# Chip-8 Emulator

A small Chip-8 emulator written in C++ using SDL3 to play some games from octo game jam in itch.io

## Build

- C++ with CMake
- Requires [SDL3](https://wiki.libsdl.org/SDL3/FrontPage)

## Run

./chip8 \<scale> \<delay> \<path_to_rom_file>

## Controles

<td>

| 1 | 2 | 3 | 4 |
|--|--|--|--|
